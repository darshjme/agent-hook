# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | ✅ Yes    |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Email **darshjme@gmail.com** with:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

You will receive a response within 48 hours.  Confirmed issues will be patched
and credited in the CHANGELOG.
